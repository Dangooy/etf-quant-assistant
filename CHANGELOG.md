# 版本说明

## v1.0.0（2026-07-07）— 首个公开版本

由五期内部开发迭代而来，每期均经独立验收与对抗性测试（详细提示词与验收标准见 docs/DESIGN-etf-weekly.md）。

### 核心能力

- **周报与交易清单**（`weekly`）：组合状态、回撤水位与断路器距离、各腿趋势/动量信号、可直接执行的交易清单（先卖后买、100 份取整、现金约束、费用估算、每条附触发规则）、存量迁移进度、上期清单执行核对、与「50%红利低波+50%国债持有不动」的对照基准线。
- **配置引擎**：趋势过滤（200 日线）→ 动量轮动（26 周，A股腿取2/海外腿取1）→ 组合回撤断路器（6% 减半 / 8% 清零并重置高水位，恢复交回趋势规则）。纯函数实现，回测与实盘信号同源。
- **组合级回测**（`backtest-portfolio`）：2016 年至今，天天基金累计净值口径（含分红、可复现），闲置现金按年化 2% 计息，支持机制对照（开/关断路器、开/关趋势过滤）。
- **风控与容错**：数据陈旧门禁（>7 天拒发清单）、持仓文件原子写入+自动备份+损坏留证、离线降级（联网失败自动用缓存并标注数据截止日）、周报心跳监控（>8 天未运行黄字告警）。
- **定时值守**：macOS launchd 模板；Windows 可用任务计划程序（见 README）。
- **AI Agent 适配**：CLAUDE.md 操作手册 + AGENTS.md 硬约束，任何模型照手册执行固定命令即可。
- **测试**：29 项 pytest 用例（撮合/费用/断路器/迁移限速/再平衡带/数据容错/指标边界）。

### 回测基准（诚实披露）

完整规则 2016-01 至 2026-07：年化 4.99%~5.11%（视现金计息假设）、最大回撤 6.21%、卡玛 0.82；2018/2022 熊市回撤 4.4%/1.9%（同期沪深300 为 28.8%/24.7%）。断路器在整个回测期从未触发（周度检查点回撤峰值 5.79%，未越过 6% 阈值），属纯保险机制。趋势过滤在样本内为负贡献（约 -1%/年），保留它是作为样本外极端熊市的保险——这是设计取舍，不是回测优化的结果。

### 已知限制

- 数据源单一（东财/天天基金），偶发 IP 限流时只能离线降级等待恢复；
- QDII 溢价保护依赖人工核对（IOPV 接口不稳定时）；
- 回测为周频调仓 + 净值口径，未模拟盘中折溢价与流动性冲击；
- 策略预期年化 5-6%，追求高收益者请绕行。

## v1.1.0（2026-07-07）— 对抗性审查加固

- **断路器日频应急清单**：组合回撤日内触及 6%/8% 时，`daily` 直接生成风险腿减仓清单（只卖不买，参考价用持仓现价），不再只是告警——堵住周度检查点之间的急跌穿透缝隙。与周度共用同一状态机，不会重复触发。
- **QDII 溢价保护**：海外腿动量选择时溢价 >3% 的标的自动跳过（防止"溢价最高时恰好动量最强"的系统性买贵）；全部超限则预算回落短融；溢价未知时保留买单但强制标注人工核对；深折价卖出附顺延提示（断路器优先级最高，仅提示不阻断）。
- **依赖锁定与季度体检**：requirements 全部锁定精确版本（干净环境复现已验证）；CLAUDE.md 新增季度体检流程（升级 akshare → 全量测试 → 周报对比 → 异常回滚）。
- 测试增至 38 项。

已知不可修风险（诚实保留）：回测期债券牛市红利不可再生、十年样本统计证据有限、免费数据源单点依赖——详见 README 的收益预期章节。
